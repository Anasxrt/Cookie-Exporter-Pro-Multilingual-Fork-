## Installation

### Firefox (Recommended - Direct Install)
Download **cookie-exporter-pro-firefox-v1.3.xpi** and install directly:
1. Open about:addons → Extensions
2. Click gear icon → Install Add-on From File
3. Select the .xpi file

### Chrome
1. Download cookie-exporter-pro-chrome-v1.3.zip
2. Extract the zip file
3. Open chrome://extensions → Enable "Developer mode"
4. Click "Load unpacked" → Select extracted chrome/ folder
